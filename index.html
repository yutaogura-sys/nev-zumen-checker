Gemini API キーの設定

図面の読み取りに Google の AI「Gemini」を使います。Google AI Studio でキーを取得して貼り付けてください。

⚠ データの取り扱い: アップロードした図面PDFは、判定のため Google の Gemini API に送信されます。個人情報・取引先名等を含む図面の取り扱いは、ご所属組織の規程に従ってください。送信データがAIの学習に利用されるかどうかは、お使いのGoogleアカウント・プランの設定によって異なります（Google AI Studio の設定をご確認ください）。APIキーは「このブラウザに保存する」を選んだ場合のみ、このPCのブラウザ内（localStorage）に平文で保存されます。共用PCでは保存しないでください。
